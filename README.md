Это система дистанционного управления пожарной машиной с двумя основными функциями:

Управление направлением брандспойта (сервоприводом)

Включение/выключение водяного насоса (через реле)

Архитектура системы:
1. Аппаратная часть:
Два микроконтроллера Arduino (один на машине, один на пульте)

Два радиомодуля nRF24L01+ для беспроводной связи

Пульт управления: джойстик + кнопка

Машина: сервопривод + реле + насос

2. Принцип работы:
На стороне ПУЛЬТА (передатчик):
Считывает положение джойстика (аналоговый сигнал 0-1023)

Считывает состояние кнопки (с защитой от дребезга)

Преобразует данные в структуру:

servoAngle: угол 0-180° (из джойстика)

pumpState: ВКЛ/ВЫКЛ (из кнопки)

Отправляет данные по радиоканалу

На стороне МАШИНЫ (приемник):
Постоянно "слушает" радиоканал

При получении данных:

Устанавливает сервопривод в заданный угол

Включает/выключает реле (насос)

Выводит статус в монитор порта для отладки

3. Ключевые технические особенности:
Связь:
Используется библиотека RF24

Одинаковая структура данных (DataPacket) на обоих устройствах

Одинаковый адрес канала ("00001")

Тип связи: однонаправленная (пульт → машина)

Управление:
Сервопривод: плавное управление через джойстик

Насос: два режима работы:

Toggle режим: кнопка-переключатель (вкл/выкл)

Мгновенный режим: насос работает только при удержании кнопки

Безопасность и стабильность:
Защита от дребезга кнопки

Ограничение угла сервопривода (0-180°)

Проверка успешности отправки данных

Рекомендации по питанию радиомодулей (3.3V)

Что происходит в реальности:
Оператор на пульте двигает джойстик и нажимает кнопку

Пульт преобразует эти действия в цифровые команды

Команды передаются по радиоканалу на расстояние до 100м (в идеальных условиях)

Машина получает команды и:

Поворачивает сервопривод (брандспойт) на нужный угол

Включает/выключает водяной насос

Вода подается в направлении, куда повернут брандспойт

Возможные улучшения:
Добавление управления движением машины

Обратная связь (статус батареи, давление воды)

Несколько каналов управления (несколько машин)

Автоматическое выключение при потере сигнала

Это базовый, но полностью рабочий прототип, который можно масштабировать под более сложные задачи робототехники и автоматизации.
